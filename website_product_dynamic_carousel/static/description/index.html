Odoo Website Dynamic Product Slider

The Dynamic Product Slider is an effective multi-purpose responsive slider with great features. A slider has multiple Slider Types, Layouts along with high quality and scalable features.

The Dynamic Product Slider can be configured with Slider Type, Layout, Product Per Slider, Product Name On/Off, Price On/Off, Quickview On/Off, Add To Cart On/Off, Autoplay On/Off and Rating On/Off.

List of Features

Drag and Drop Snippet with dynamic configuration.

Create your multiple product collections.

Multiple Slider Type Options. (Single Tab Slider, Multi Tab Slider)

Multiple Layout Options. (Slider, Full-width Slider, Grid, Horizontal Tab Lauout, Vertical Tab Layout etc...)

Choose an individual product collection for the slider.

Slider configuration with dynamic filters like Product Per Slider, Product Name On/Off, Price On/Off, Autoplay On/Off etc...
